Bash Shortcut

Jan 2nd, 2015 2:15 am

lazy 快捷鍵

  • ctrl + a: 將游標移到開頭
  • ctrl + e: 將游標移到結尾
  • ctrl + k: 刪除游標後面的所有字元
  • ctrl + u: 刪除游標前面的所有字元
  • ctrl + w: 刪除游標前面的所有字元
  • ctrl + t: 將游標前的兩個字母對調
  • ctrl + c: 中斷執行
  • cmd + t: 開一個新的頁籤
  • cmd + n: 開一個新 bash
  • tab: 自動完成檔案或目錄名稱
